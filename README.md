# one-click-tar README

## Features

This is a ultra-simple extension that I made, just because I did not want to remember tar command, or move to directory with cd each time I need to tar. 

When you right click a directory, you would see a new option to create .tar file. When you click it, it makes a tar file in the same destination as the target directory.

## Requirements

The `tar` command must be available in the environment where the extension runs. Linux most certainly has it, but windows may not, if in the old version. 

- For local use, `tar` must be available on your local machine.
- For Remote-SSH use, `tar` must be available on the remote server.
- On modern Windows, `tar` is usually available by default.
- On Linux, `tar` is usually available by default, but it may need to be installed depending on the environment.
